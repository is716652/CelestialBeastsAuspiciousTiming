## 一、这是什么？—— **二十八宿气象占法**

### ✅ 核心定义：

> 利用 **当日干支（日辰）的五行** 与 **当日值宿（日宿）的五行/属性** 之间的生克关系，结合 **季节（春、夏、秋、冬）**，推断当日或近期的 **天气变化（晴、雨、风、雾、雪等）**。

这是一种**将星象、五行、节气融合的古代气象预报模型**，在农耕、行军、祭祀、出行中具有极高实用价值。

------

## 二、核心原理（三层逻辑）

### 🌟 第一层：**日辰 vs 日宿 五行生克**

| 关系                           | 天气                   |
| ------------------------------ | ---------------------- |
| **日辰克日宿**（如辰水克宿火） | 阴晦、多云             |
| **日宿克日辰**（如宿水克辰火） | 天晴（水克火反主明朗） |
| **日辰生日宿**（如辰木生宿火） | 阴天                   |
| **日宿生日辰**（如宿木生辰火） | 晴                     |
| **比和**（同五行）             |                        |

- 皆火 → 晴  
- 皆金/水 → 雨  
- 皆土 → 阴晦  
- 皆木 → 风

> 🔍 **例**：
> 甲子日（纳音海中金，属金），值宿为“毕月乌”（水禽）→ 金生水 → **日辰生日宿** → 主阴。

------

### 🌟 第二层：**二十八宿本象属性**

每宿有其 **自然象征**，直接对应天气：

| 星宿               | 象征            | 天气倾向   |
| ------------------ | --------------- | ---------- |
| **角、亢**         | 木 → 风         | 狂风、砂尘 |
| **氐、房、心、尾** | 火/木 → 热+风   | 雨风交加   |
| **箕、斗**         | 水 → 风雨       | 风浪、暴雨 |
| **牛、女**         | 土/金 → 收敛    | 先晴后雨   |
| **虚、危**         | 水 → 寒         | 薄暮见晴   |
| **室、壁**         | 水 → 寒湿       | 多风雨     |
| **奎、娄**         | 木/金 → 清肃    | 多晴       |
| **胃、昴、毕**     | 土/金/月 → 光明 | 雨后转晴   |
| **觜、参**         | 火/水 → 冲突    | 飒飒风雨   |
| **井、鬼、柳**     | 木/金/土 → 混浊 | 阴晦       |
| **星、张、翼、轸** | 火/月/水 → 变化 | 晴雨不定   |

------

### 🌟 第三层：**分季节细化（春、夏、秋、冬）**

同一星宿在不同季节，天气表现不同：

| 季节           | 特点         | 举例                                         |
| -------------- | ------------ | -------------------------------------------- |
| **春**（木旺） | 风多、雨渐生 | 角亢主狂风，奎娄反主晴（木克土，土晴）       |
| **夏**（火旺） | 炎热、骤雨   | 鬼柳大雨（火土相激成雨），昴毕晴（金制火）   |
| **秋**（金旺） | 干燥、风起   | 奎娄胃昴雨（金生水），星张翼轸晴（火被金制） |
| **冬**（水旺） | 寒冷、雪雾   | 参井雨（水旺），鬼柳星张晴（火土暖局）       |

> 📌 **关键**：
> “**春雨贵如油，夏雨急如箭，秋晴高爽，冬雪封山**” —— 气象占必须结合季节！
