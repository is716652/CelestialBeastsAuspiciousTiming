> 伏断日 
>
> 宜作厕陂塘塞穴断蚁獭，余事俱凶。 子虚丑斗寅嫌室，卯女辰箕巳怕房。 午角未张申见鬼，酉觜戌胃亥壁当。 伏断诗诀 太阳值日子寅酉，太阴值日午上寻。 火居卯丑戌，水未土在巳，金申木亥辰。



------

### 一、**基本定义与用途**

伏断日指**天干地支与二十八宿中的特定星宿配合**的日子，传统认为此日适合进行某些特定活动，但其他事务则多有禁忌。其主要用途包括：

- **宜**：断奶、堵塞鼠穴、驱除白蚁、修建厕所或池塘等。
- **忌**：出军、出行、嫁娶、上任、修造建筑等重要活动。

------

### 二、**推算方法**

伏断日的推算基于**二十八宿与十二地支的对应关系**，具体规则如下：

#### 1. **地支与星宿的固定对应**

根据《增补万全玉匣记》等古籍记载，十二地支每日对应的星宿为：

| 地支 | 对应星宿 |
| ---- | -------- |
| 子   | 虚宿     |
| 丑   | 斗宿     |
| 寅   | 室宿     |
| 卯   | 女宿     |
| 辰   | 箕宿     |
| 巳   | 房宿     |
| 午   | 角宿     |
| 未   | 张宿     |
| 申   | 鬼宿     |
| 酉   | 觜宿     |
| 戌   | 胃宿     |
| 亥   | 壁宿     |

**举例**：若某日地支为“子”，且当日值宿为虚宿，则该日为伏断日。

#### 2. **七政（日月金木水火土）与时辰的配合**

伏断日的时辰选择需结合七政值日：

- **太阳值日**：子时、寅时、酉时
- **太阴值日**：午时
- **火星值日**：卯时、丑时、戌时
- **水星值日**：未时
- **土星值日**：巳时
- **金星值日**：申时
- **木星值日**：亥时、辰时

**举例**：若某日为太阳值日，则子时、寅时、酉时为伏断时。

### 三、**实际应用与争议**

1. **民间习俗**
   伏断日最著名的应用是**婴儿断奶**，民间认为此日断奶可减少婴儿哭闹，避免健康问题。例如2025年5月的伏断日包括1日、2日、5日、9日等。
2. **科学争议**
   现代观点认为，伏断日属于**传统择吉文化中的迷信范畴**，其吉凶判断缺乏科学依据。断奶等事务更应依据医学建议（如婴儿发育阶段、季节适宜性）而非择日。

### 四、**伏断日诗诀解析**

用户提供的诗诀可拆解为两部分：

1. **地支星宿对应**：
   “子虚丑斗寅嫌室，卯女辰箕巳怕房……”
   即通过地支与星宿的固定对应关系确定伏断日。
2. **七政时辰配合**：
   “太阳值日子寅酉，太阴值日午上寻……”
   指不同天体值日时，伏断时辰的优选时段。
