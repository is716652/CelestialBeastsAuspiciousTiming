这是一个非常专业且关键的问题。

在演禽术体系中，“**七曜贵人禄星**”与“**禽星贵人宿**”虽然都涉及“贵人”概念，但**来源不同、逻辑不同、用法也不同**。前者源于**七政四余（天文星占）**，后者源于**二十八宿禽形类象**。

下面为你系统梳理两者的完整内容与本质区别：

------

## 一、七曜贵人禄星（源自七政四余）

“七曜”指日、月、火、水、木、金、土七星（即七政），在演禽术中被赋予**人格化神性**，每曜有其**贵人、禄神、福星**等职能。

> 📌 **核心逻辑**：以**日干**或**命主本命**为基准，查七曜所临之位，定贵人禄马。

### ✅ 1. 七曜贵人禄星完整表（按日干起例）

| 日干   | 贵人曜   | 禄神曜   | 福星曜   | 所属禽宿（演禽对应）                                     |
| ------ | -------- | -------- | -------- | -------------------------------------------------------- |
| **甲** | **土星** | **火星** | **木星** | 土：氐、女、柳、胃 火：尾、房、觜、翼 木：角、斗、井、奎 |
| **乙** | **木星** | **金星** | **水星** | 木：同上 金：亢、牛、娄、鬼 水：箕、壁、参、轸           |
| **丙** | **水星** | **木星** | **金星** | ——                                                       |
| **丁** | **金星** | **水星** | **土星** | ——                                                       |
| **戊** | **土星** | **火星** | **木星** | 同甲                                                     |
| **己** | **木星** | **金星** | **水星** | 同乙                                                     |
| **庚** | **火星** | **土星** | **水星** | ——                                                       |

> 🔔 注：此表仿《七政四余》贵人禄马起法，在演禽中需将七曜**转化为对应禽宿**。

### ✅ 2. 七曜贵人禄星的判断方式

- **贵人**：主得人扶持、官非化解。
- **禄神**：主钱财、俸禄、收益。
- **福星**：主平安、喜庆、灾厄减免。
- **用法**：若排盘中**时禽/日禽**为当日贵人曜所化禽星，则主贵人至。

**例**：
庚日占事，贵人为**火星** → 火禽为尾（虎）、房（兔）、觜（猴）、翼（蛇）。
若时禽为“尾火虎”，则**贵人临事**，主得武职之人相助。

## 二、禽星贵人宿（源自二十八宿禽形）

如前所述，这是基于**禽兽习性与地支方位**的类象系统，不依赖日干，而看**禽星是否入庙、得时、合局**。

| 星宿   | 贵人类象   | 应验条件       |
| ------ | ---------- | -------------- |
| 井木犴 | 权臣、帝王 | 入未宫、四季末 |
| 奎木狼 | 文官、师长 | 入戌宫、春季   |
| 星日马 | 驿使、上司 | 入午宫、夏季   |
| 毕月乌 | 监察、法官 | 入申宫、秋季   |
| 虚日鼠 | 谋士、密友 | 入子宫、冬季   |

> 📌 **核心逻辑**：**禽性 + 方位 + 时令 = 贵人能量**

## 三、本质区别对比表

| 维度               | **七曜贵人禄星**       | **禽星贵人宿**               |
| ------------------ | ---------------------- | ---------------------------- |
| **来源**           | 七政四余（天文星曜）   | 二十八宿禽形类象             |
| **起例依据**       | **日干**（或命主八字） | **禽星自身属性**（无需日干） |
| **核心变量**       | 日干 → 七曜 → 禽宿     | 禽宿 → 地支 → 旺衰           |
| **适用场景**       | 命理推运、长期运势     | 即时占卜、具体事件           |
| **是否需出生时间** | 是（需日干）           | 否（仅需问事时辰）           |
| **贵人类型**       | 抽象神性（贵人/禄/福） | 具象角色（文官/武将/驿使）   |
| **典型用法**       | “甲日见土禽为贵人”     | “井犴入未，主权威贵人”       |

## 四、实战中如何结合？

在高级演禽术中，二者可**叠加使用**：

> **步骤**：  
>
> 1. 先看**禽星贵人宿**：盘中是否有井犴、奎狼等入庙？→ 判断**是否有贵人**。  
> 2. 再看**七曜贵人**：若问事者日干为庚，而盘中见尾火虎（火禽）→ 此贵人**正合日干所需**，力量倍增。

**例**：  

- 占官司，时禽为“毕月乌”（监察贵人），翻禽为“鬼金羊”。  
- 若问事者日干为**丁**，丁日贵人为**金星** → 鬼金羊为金禽 → **贵人+用神合一** → 必胜。
