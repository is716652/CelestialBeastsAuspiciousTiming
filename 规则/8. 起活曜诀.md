> 房虚昴星四日宿翻毕月乌，心危毕张四月宿翻尾火虎，尾室觜翼四火宿翻奎木狼，箕壁参轸四水宿翻氐土貉，斗奎角井四木宿翻虚日鼠，亢牛娄鬼四金宿翻牛金牛，氐女胃柳四土宿翻箕水豹。 用值时禽星，如是日太阳加寅上起，则翻毕，或是火则翻奎，将奎加寅上逆寻时禽上止，顺回时上即为活曜。 切记每时不同，如子虚日鼠，即翻毕于寅；如丑危月燕，即翻尾于寅；如室火猪，即翻奎于寅，余仿此。 假如丁卯日，壁水渝值日，午时用事，时禽是星日马，日则翻毕，将毕加寅上，逆寻星日马在申，就以星日马从申上顺寻，午时是箕水豹，名为活曜，为我禽也，余仿此。



这是一段非常核心且进阶的演禽口诀，名为**“起活曜诀”**（也称“倒将”）。它与上一节的“翻他禽”虽然都叫“翻”，但**逻辑、方向和用途完全不同**。

为了让你彻底搞懂，我将先解释这段口诀的含义，然后再详细对比它与上一节“翻他禽”的区别。

### 📜 第一部分：解释“时禽翻活曜诀”

**核心概念：**

- **活曜（倒将）：** 代表**“我”**的最终落点、变化或隐藏的状态。
- **逻辑：** 它不是去找“对方”，而是根据“日禽”的属性，定下一个固定的“翻转起点”（寅位），然后通过“逆数找时禽，顺数回时辰”的方式，找出一个新的星宿，这个星宿就是“活曜”（真正的我）。

#### 🛠️ 推算步骤（拆解口诀）

1. **定“翻转起点”（寅位定宿）：**
   根据当天**日禽**的五行属性，确定一个固定的星宿，把这个星宿加在**“寅”**这个地支上。
   - **日/月/火/水/木/金/土** -> 对应 **毕/尾/奎/氐/虚/牛/箕**。
   - *口诀：日毕月尾火翻奎，水氐木虚金骑牛，土宿还从箕位起。*
2. **逆寻时禽：**
   从第一步放在“寅”位的那个星宿开始，**逆时针**数，一直数到**“时禽”**（即之前算出的代表时辰的星宿）为止。
3. **顺回活曜：**
   找到时禽后，以此为起点，**顺时针**往回数，数回到**“用事之时”**（比如午时）。
   - 此时落在“时”位上的星宿，就是**“活曜”**。

#### 📖 实例演示（口诀中的例子）

- **条件：** 丁卯日，日禽是**壁水渝**（属水），午时用事，时禽是**星日马**。
- **步骤：**
  1. **定起点：** 日禽“壁水渝”属**水**。根据口诀“水氐”，属水的日子，翻转起点是**氐土貉**。将“氐土貉”加在**寅**上。
  2. **逆寻时禽：** 从寅位的“氐”开始，**逆时针**数，一直数到找到**“星日马”**为止（假设数到了某个地支）。
  3. **顺回活曜：** 从“星日马”所在的位置，**顺时针**数，数回**午时**。
  4. **结果：** 数到午时上是**箕水豹**，这个“箕水豹”就是**活曜**（我禽）。

### ⚔️ 第二部分：对比“翻他禽”与“翻活曜”的区别

这是演禽术中最容易混淆的两个概念，也是你问到的“对比上文的区别”。简单来说，一个是**找敌人**，一个是**变自己**。

| 比较维度     | **翻他禽（上一节内容）**         | **翻活曜/倒将（本节内容）**                         |
| ------------ | -------------------------------- | --------------------------------------------------- |
| **别名**     | 天禽、彼禽、他人                 | 活曜、倒将、我禽                                    |
| **核心目的** | **寻找对手/环境**                | **寻找真实的自己/变化**                             |
| **逻辑起点** | 以“将星”（日/年符头）加在“时”上  | 以“日禽属性”定死“寅”位                              |
| **运行轨迹** | **顺数**找落点 -> **逆数**回“时” | **逆数**找时禽 -> **顺数**回“时”                    |
| **代表意义** | **体用关系** 看对方对我怎么样    | **主客转化** 看我在这件事中如何应变、隐藏或最终结果 |
| **通俗比喻** | 派侦察兵去敌营，看敌人是谁       | 看镜子，看自己换上伪装后的样子                      |

### 📌 总结与应用

在演禽排盘中，这三者构成了完整的“一课”：

1. **时禽（地盘）：** 原本的我，事情的基础。
2. **翻他禽（天盘/彼）：** 对手或环境，用来判断**“体用生克”**（我和对方的关系）。
3. **翻活曜（倒将/我）：** 变化的我，用来判断**“吉凶成败”**（我能不能搞定这件事）。

**为什么叫“倒将”？**
因为它的数法是“逆寻顺回”，与正常的顺时针运行（正将）相反，是“倒着走”的，所以叫倒将。这代表了事情的**转机、隐藏、曲折**。

**一句话总结：**
上一节的“翻他禽”是**向外看**，找出竞争对手；这一节的“翻活曜”是**向内求**，找出自己应对这件事的最佳状态或最终结果。两者结合，才能断事如神。
