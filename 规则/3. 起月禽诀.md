> 起月禽诀 
>
> 
>
> 会得年禽月易求，太阳用角木参头。 太阴起室火星取，金心土胃水骑牛。 其法如太阳值年，正月起角宿，顺行十二月。 又如康熙壬辰年系毕月乌值年，月系太阴，正月便是室宿值月，二月壁宿值月，顺轮十二月。



这是一段关于**演禽术**中如何推算**“月禽”**（即每月值班的星宿）的核心口诀。

简单来说，**“年禽”\**决定了\**“月禽”\**的起始点。你只需要知道今年是哪个星宿值班（年禽），再根据这个年禽的\**五行属性**（金木水火土日月），就能确定正月从哪个星宿开始数。

我来为你详细拆解这段口诀的含义和用法：

### 📜 核心口诀解析

> **口诀：** “会得年禽月易求，太阳用角木参头。太阴起室火星取，金心土胃水骑牛。”

这句口诀其实是一个**“查表规则”**，它规定了不同属性的年份，正月的起始星宿是不同的：

| 年禽的属性     | 口诀对应     | 正月起始星宿 | 备注                             |
| -------------- | ------------ | ------------ | -------------------------------- |
| **太阳（日）** | **太阳用角** | **角木蛟**   | 属“日”的年份，正月从角宿开始。   |
| **太阴（月）** | **太阴起室** | **室火猪**   | 属“月”的年份，正月从室宿开始。   |
| **火星（火）** | **火星取**   | **室火猪**   | 属“火”的年份，正月也从室宿开始。 |
| **金星（金）** | **金心**     | **心月狐**   | 属“金”的年份，正月从心宿开始。   |
| **土星（土）** | **土胃**     | **胃土雉**   | 属“土”的年份，正月从胃宿开始。   |
| **水星（水）** | **水骑牛**   | **牛金牛**   | 属“水”的年份，正月从牛宿开始。   |
| **木星（木）** | **木参头**   | **参水猿**   | 属“木”的年份，正月从参宿开始。   |

------

### 🧮 推算方法（步骤）

一旦确定了当年的“年禽”（比如今年是“虚日鼠”），请按以下步骤推算各月的月禽：

1. **定属性：** 看看今年的年禽属于哪一行（七政）。比如“虚日鼠”带“日”字，属于**太阳（日）**；“毕月乌”带“月”字，属于**太阴（月）**。
2. **找起点：** 根据上表，找到该属性对应的**正月起始宿**。例如：如果是“日”属性，正月就从**角宿**开始；如果是“月”或“火”属性，正月就从**室宿**开始。
3. **顺时针数：** 从正月的起始宿开始，按照二十八宿的固定顺序（角、亢、氐、房、心、尾、箕、斗、牛、女、虚、危、室、壁、奎、娄、胃、昴、毕、觜、参、井、鬼、柳、星、张、翼、轸），**顺数**，一个月数一个宿。

------

### 📖 实例讲解

你提供的例子里提到：“康熙壬辰年系毕月乌值年，月系太阴，正月便是室宿值月……”

- **分析：** “毕月乌”属于**太阴（月）**。
- **查表：** 根据口诀“太阴起室”，所以正月从**室火猪**开始。
- **推算：**
  - 正月：室火猪
  - 二月：壁水渝（室宿的下一个）
  - 三月：奎木狼（壁宿的下一个）
  - …以此类推，直到十二月。

------

### ⚠️ 特别注意（进阶知识）

虽然古籍中普遍流传上述口诀，但在实际演禽排盘中，**月禽的起法**有时会根据流派不同而有所调整。

1. **“番禽”与“活曜”：** 有些演禽流派（如七元演禽）中，月禽不仅用于看当月吉凶，还用于起“番禽”（飞禽）和“活曜”，这时候可能会涉及到更复杂的加减规则。
2. **历法修正：** 现代有研究指出，古籍中的月禽起法可能存在“错位”或“冬至起算”的特殊情况（即起点可能不是正月，而是冬至所在的十一月），但在基础的“太阳、太阴、金木水火土”分类起法中，上述口诀是通用的标准方法。

**总结：** 这首诀窍就是教你**“看年份属性，定正月星宿”**。只要记住“日从角、月火从室、金从心、土从胃、水从牛、木从参”，就能排出全年的月禽了。
