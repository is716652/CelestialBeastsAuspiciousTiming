



**禽星锁泊**是演禽术中极为关键的一环，它解决的是**“地利”**的问题。

简单来说，**“锁泊”\**就是给禽星（星宿）找一个\**“落脚点”**。排出了年、月、日、时禽之后，这只是有了“人”（禽星），但这个人是在山上、在水里、还是在锅里（环境），这就需要通过“锁泊”来确定。

你提到的**“张良锁泊歌”**（即锁泊口诀）正是用来判断这个“落脚点”吉凶的核心依据。

以下为你详细拆解这三个概念：

### 🔒 什么是“禽星锁泊”？

**字面意思：**

- **锁：** 锁定。
- **泊：** 停船、停靠。
- **合起来：** 就是看这只“禽”（星宿）飞累了，落在了哪里（十二地支盘上的哪个位置）。

**实战意义：**
它主要用来判断**禽星得地不得地**。

- **得地（吉）：** 比如“龙归大海”、“虎入深山”，力量强大，能发挥好的作用。
- **不得地（凶）：** 比如“龙陷浅水”、“虎落平阳”，力量受制，容易出凶险。

------

### 🗺️ 锁泊十二宫（内盘与外盘）

在锁泊时，演禽术使用的是**十二地支盘**，但每个地支对应了不同的地理环境。通常分为**“内盘”**（大环境/长久）和**“外盘”**（小环境/暂时）。

**十二地支对应环境表（锁泊基础）：**

| 地支   | 内盘（大环境）  | 外盘（小环境） |
| ------ | --------------- | -------------- |
| **寅** | 山              | 山             |
| **卯** | 林              | 水             |
| **辰** | 岗（山冈）      | 田             |
| **巳** | 汤（热水/温泉） | 园             |
| **午** | 火              | 刀             |
| **未** | 野              | 砧（切肉的板） |
| **申** | 刀              | 路             |
| **酉** | 砧              | 江             |
| **戌** | 路              | 湖             |
| **亥** | 江              | 田             |
| **子** | 湖              | 野             |
| **丑** | 田              | 火汤           |

**排法口诀：**

> **“山水田园井，刀天草岸风，火月周流转。”**

**起盘方法：**
根据**时禽**（或日禽）的五行属性（七政），在特定的地支起“山”字，然后顺时针数，数到**用事之时**（比如午时），看落在哪个字上。

- **例如：** 火星（火禽）起寅为山，卯为水，辰为田……数到午时，午上是“火”或“刀”。

### 🔄 禽星变禽法（锁泊格局）

当禽星落在不同的“泊地”时，它的性质会发生变化，甚至会“变身”。这就是**“变禽法”**。

比如：**“蛟龙”\**如果落在水里，可能变成\**“真龙”**；如果落在火里，可能变成**“死鱼”**。

**常见变禽与格局（张良锁泊歌的应用）：**

这里引用你提到的**“张良锁泊歌”**（即古籍中流传的格局断语）来解释：

1. **角木蛟 / 亢金龙**
   - **泊亥子（江湖）：** **极乐宫**。龙归大海，兴云布雨，大吉。变身为“活龙”。
   - **泊寅（山）：** 为“**龙虎争珠**”，主争斗、是非。
   - **泊午（火/汤）：** 为“**龙陷汤镬**”或“**龙奔枯井**”。龙怕火，大凶，主受人欺压、做事艰辛。
   - **泊戌（路）：** 为“**蛟龙失水**”或“**贵入贱宫**”，主运势退散。
2. **尾火虎 / 箕水豹**
   - **泊寅卯（山林）：** **喜宫**。虎入深山，豹入丛林，得地，大吉。
   - **泊亥（江）：** 为“**猛虎出林**”或“**渡江**”，虽有险但有贵气。
   - **泊午（火）：** 为“**虎遭火焚**”，大凶。
3. **牛金牛**
   - **泊午（火）：** 为“**耕石田**”，主做事艰辛，百无一成。
   - **泊寅（山）：** 为“**负重山**”，出行大忌，做事艰难。
4. **心月狐**
   - **泊寅（山）：** 为“**狐假虎威**”，宜托人办事，借力使力。

### 📜 张良锁泊歌（核心口诀解析）

你提到的**“张良锁泊歌”**，通常指的就是上述这些**“某某禽泊某某地为某某格”**的断语集合。它是演禽术的**“象法”**基础。

**它的逻辑是：**

1. **看五行生克：** 水禽泊水地（得地），火禽泊水地（受制）。
2. **看物理特性：** 鱼在水里好（乐宫），鱼在火里烤（死宫）。
3. **看特殊格局：** 比如“玉兔守株”、“金钗落井”等形象比喻。

### 📌 总结：演禽断事的完整流程

1. **起禽：** 算出年、月、日、时禽（谁来了？）。
2. **翻禽：** 算出翻禽（对手是谁？）。
3. **锁泊：** 给这些禽星找落脚点（他们在哪儿？环境如何？）。
4. **变禽：** 根据落脚点判断他们变成了什么状态（是龙是虫？）。
5. **定吉凶：** 结合生克（比如我禽克他禽为胜，他禽生我禽为利）。

所以，**“锁泊”\**就是给干巴巴的星宿赋予了\**“场景感”**，让占卜结果更加生动具体（比如断出行：如果是“鸟泊刀砧”，那可能就是路上有灾；如果是“鸟泊风月”，那可能就是路上有艳遇）。





结合你刚才提供的**“演禽盘式”**（包含地禽、翻禽、活曜的推算）和现在的**“禽星锁泊”**，这两者其实是演禽术中**“排盘”**与**“断局”**两个截然不同的阶段。

简单来说：**演禽盘式是“起卦”，禽星锁泊是“解卦”。**

为了让你彻底理清这两者的区别，我为你整理了一个详细的对比分析：

### 📖 详细解析

#### 🧮 演禽盘式：这是“骨架” (怎么排？)

这是你之前发的那个长篇内容。

- **它在做什么？** 它是在做**加减乘除**。
- **流程：** 通过年、月、日、时的干支，利用“顺数”、“逆数”、“翻转”等数学规则，从二十八宿中筛选出几个关键的星宿。
- **结果：** 它告诉你：**“我”是毕月乌，“对方”是张月鹿，“变数”是井木犴。**
- **特点：** 这是一个**动态**的过程，每换一个时间，排出来的星宿就会变。

#### 🔒 禽星锁泊：这是“血肉” (怎么看？)

这是你刚才问的“锁泊12宫”、“变禽法”。

- **它在做什么？** 它是在做**翻译和比喻**。
- **流程：** 当你通过“盘式”算出了“毕月乌”（我）之后，你还要看“毕月乌”现在落在了哪里（十二地支的哪个位置）。
  - 如果“毕月乌”落在了**亥子**（水），那就是“鸟投江”，可能代表事情泡汤或人落水。
  - 如果“毕月乌”落在了**寅卯**（木），那就是“鸟宿林”，代表安稳、归宿。
- **结果：** 它告诉你：**“毕月乌”现在处于“鸟投江”的凶险状态，或者“鸟宿林”的安稳状态。**
- **特点：** 这是一个**静态**的查表过程（查口诀），用来判断吉凶的具体细节。

------

### ⚦ 两者是如何结合的？（实战演示）

假设你要算一卦“外出求财”，时间是“庚申日、庚辰时”。

#### 第一步：演禽盘式（排盘）

- 你通过复杂的计算（顺数、翻转），得出了**“地禽（我）”是“角木蛟”**。

#### 第二步：禽星锁泊（断局）

- 现在你知道了“我”是“角木蛟”（龙），但这龙是好是坏？
- 你就要用**“锁泊”**：
  - 看这个“角木蛟”落在了十二地支的哪个宫位。
  - **如果是落在“亥子”（水）：** 锁泊歌曰“龙归大海”。**大吉！** 说明你如鱼得水，求财顺利。
  - **如果是落在“午”（火）：** 锁泊歌曰“龙陷汤镬”（龙掉进了开水锅）。**大凶！** 说明你会遇到灾祸，不仅没财，还可能受伤。

### 📌 总结

- **演禽盘式**是**“找演员”**（确定谁出场了）。
- **禽星锁泊**是**“搭舞台”**（确定他们在舞台上发生了什么事）。

没有盘式，你不知道谁在演；没有锁泊，你不知道演得怎么样。两者结合，才是一个完整的演禽预测过程。
