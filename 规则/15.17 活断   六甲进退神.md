



演禽术（或融合六壬、择日术）中极为精微的 **“六甲进退神”系统**，用于判断 **每日气机的动态趋势**——是“进”（发展）、“退”（消散）还是“伏”（潜藏），从而决定 **出行、求财、战斗、治病、嫁娶等事项的吉凶成败**。

下面我将 **系统拆解、逐层讲解，并配以清晰表格**，助您掌握这一高阶技法。

------

## 一、核心概念：进神、退神、伏神

| 神煞     | 象义               | 适用事体               |
| -------- | ------------------ | ---------------------- |
| **进神** | 气机上升、事态推进 | 求财、远行、求官、签约 |
| **退神** | 气机下降、事态收敛 | 治病、退敌、解约、静守 |
| **伏神** | 气机潜藏、事态停滞 | 战斗、埋伏、暗谋、避祸 |

> ✅ **古诀**：“五日一进好求财，五日一退好治病，五日一伏好战斗。”

## 六、实战应用法则

### ✅ 1. **择日用神**

- **求财、签约、远行** → 选 **进中进** 日（如甲子、甲午、己酉起头的5日）。
- **治病、退敌、解约** → 选 **退中退** 日。
- **战斗、埋伏、暗谋** → 可用 **伏神日**（但需极谨慎，因后段常接“天空亡”）。

### ⚠️ 2. **避凶原则**

- **凡遇“天空亡”**（即伏神日后段）→ **百事忌用**，主无果、破财、灾祸。
- **进中退** → 事初成后败，不宜重大决策。

### 🧭 3. **与禽星结合**

- 若日禽为 **强禽（如井犴、尾虎）** + **进中进** → **大吉加倍**。
- 若日禽为 **无头星** + **伏神日** → **凶上加凶**。
