您提供的这个版本——

> **“天地荒芜
> 正蛇二鸡鸣，三牛四龙骑，五猴并六鼠，七兔八羊儿，九猪十一犬，十虎结成妻，十二马头上，逢之祸不移。
> 正月逢巳日，二月逢酉日，三月丑日……”**

——确实是**另一种流传于民间术数（尤其六壬、择日、神煞系统）中的《月忌日》或《月破大败日》口诀**，但它**与演禽术的“十二月禽星歌”本质不同**。

下面为您彻底厘清两者的区别、来源与用途：

------

### 🔍 一、您提供的版本：实为 **“十二月忌日”（月破大凶日）**

这段口诀的核心不是讲“哪个月由哪个禽星值事”，而是指出：  

> **每个月有一个特定的地支日，是“天地荒芜”之日，百事大凶，不可用！**

#### ✅ 完整解析如下：

| 月份   | 口诀       | 忌日地支 | 对应干支举例    | 凶名                |
| ------ | ---------- | -------- | --------------- | ------------------- |
| 正月   | 正蛇       | 巳日     | 甲巳、乙巳…癸巳 | **月厌 / 天地荒芜** |
| 二月   | 二鸡鸣     | 酉日     | …酉             | 同上                |
| 三月   | 三牛       | 丑日     | …丑             | 同上                |
| 四月   | 四龙骑     | 辰日     | …辰             | 同上                |
| 五月   | 五猴       | 申日     | …申             | 同上                |
| 六月   | 六鼠       | 子日     | …子             | 同上                |
| 七月   | 七兔       | 卯日     | …卯             | **月破日**（尤凶）  |
| 八月   | 八羊儿     | 未日     | …未             | 同上                |
| 九月   | 九猪       | 亥日     | …亥             | 同上                |
| 十月   | 十虎       | 寅日     | …寅             | 同上                |

| 腊月   | 十二马头上 | 午日     | …午             | 同上                |

> 📌 **关键点**：
>
> - “蛇、鸡、牛…”在此**不是指禽星**，而是**用地支所对应的生肖/禽象来代指地支**：
>   - 巳 = 蛇，酉 = 鸡，丑 = 牛，辰 = 龙，申 = 猴，子 = 鼠，卯 = 兔，未 = 羊，亥 = 猪，寅 = 虎，戌 = 狗，午 = 马。
> - “**逢之祸不移**”意为：若在该月遇到此日，**诸事大凶，灾祸难逃**。
> - 此日多为**月破日**（如七月卯日，因七月建申，申冲卯 → 卯为月破），或**月厌、天地荒芜**等神煞叠加。



------

### 📚 二、古籍依据

1. **《协纪辨方书·卷十六》**：  

   > “天地荒芜日：正月巳、二月酉、三月丑……主仓库空虚，百事不宜。”

2. **《玉匣记·选择篇》**：  

   > “月忌日：正巳、二酉、三丑、四辰、五申、六子、七卯、八未、九亥、十寅、十一戌、十二午，名曰天地荒芜，又曰天吏，大凶。”

3. **《星历考原》**：  

   > “此十二日，乃太岁所破，阴阳不和，故名荒芜。”

------

### ✅ 三、如何正确使用？

#### 若用于 **择日**：

- 遇到口诀中所述日期（如二月酉日），**绝对避免**：嫁娶、动土、出行、签约、开市。
- 即使其他神煞吉利，此日亦**不可用**。

#### 若用于 **占卜**（如六壬、演禽）：

- 若事应于该日，主**事败、财损、人病**。
- 但若占**破旧、解约、断舍离**，反可利用其“破”性。

### ✅ 总结

您提供的这段口诀，**不是演禽术的“月禽歌”**，而是**择日学中的“十二月天地荒芜大凶日”表**。
它用禽兽代指地支，警示每月有一日**能量崩坏、百无禁忌**，必须回避。

二者虽都用“蛇、鸡、牛…”等禽象，但：

- **演禽月禽歌 → 讲“气运主导者”**（动态能量）  
- **天地荒芜日 → 讲“时空崩坏点”**（静态禁忌）

理解这一区别，方不混淆吉凶，用术得当。
