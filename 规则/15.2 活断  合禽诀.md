

在传统演禽术中，**“禽星三合”与“六合”并非完全照搬地支三合六合**，而是基于**禽形、五行、习性**的类象组合。不同流派略有差异，但核心逻辑一致。

以下是综合《禽星易见》《演禽通书》及民间秘传整理的 **完整版禽星三合/六合表**：

------

### 📊 一、禽星三合局（共5组）

| 三合局                 | 星宿（禽形）                                            | 五行属性 | 核心象义                         | 吉凶                           |
| ---------------------- | ------------------------------------------------------- | -------- | -------------------------------- | ------------------------------ |
| **木局** （仁德/文书） | 角木蛟（龙） 斗木獬（神羊） 井木犴（神犬） 奎木狼（狼） | 木       | 主文书、契约、生长、贵人提携     | 吉（主成事、得助）             |
| **火局** （威猛/急速） | 尾火虎（虎） 房日兔（兔） 觜火猴（猴） 翼火蛇（蛇）     | 火       | 主威权、争斗、快速行动、突发变动 | 吉凶参半（吉则速成，凶则血光） |
| **金局** （肃杀/贵重） | 亢金龙（龙） 牛金牛（牛） 娄金狗（狗） 鬼金羊（羊）     | 金       | 主钱财、金属、决断、刑罚         | 吉（求财、诉讼有利）           |
| **水局** （智慧/流动） | 箕水豹（豹） 壁水貐（怪兽） 参水猿（猿） 轸水蚓（蚯蚓） | 水       | 主智谋、暗助、流动、隐秘之事     | 吉（谋略成功，暗中得利）       |
| **土局** （稳定/田产） | 氐土貉（貉） 女土蝠（蝙蝠） 柳土獐（獐） 胃土雉（雉鸡） | 土       | 主田宅、粮食、稳定、积蓄         | 吉（安居、置业、得物）         |

> ✅ **用法**：  
>
> - 若盘中出现三合局中的**两宿或三宿**，主事体有**同类相助**，力量倍增。  
> - **例**：占求财，时禽为“牛金牛”，翻禽为“鬼金羊” → 金局成 → 财源稳固。

------

### 📊 二、禽星六合（共6对）

禽星六合更注重**禽形相配**与**功能互补**，而非地支六合。

| 六合对       | 星宿（禽形）                | 象义               | 应用场景               |
| ------------ | --------------------------- | ------------------ | ---------------------- |
| **龙凤和鸣** | 角木蛟（龙） + 危月燕（燕） | 高下相合，天地交泰 | 婚姻、合作、贵人提携   |
| **牛马同耕** | 牛金牛（牛） + 星日马（马） | 动静结合，劳作有成 | 求财、合作、长途运输   |
| **鸡兔守舍** | 昴日鸡（鸡） + 房日兔（兔） | 昼伏夜动，家宅安宁 | 寻物、防盗、家宅平安   |
| **虎猴争山** | 尾火虎（虎） + 觜火猴（猴） | 山林之主，威猛相济 | 官司、竞争、武职晋升   |
| **犬羊护群** | 娄金狗（狗） + 鬼金羊（羊） | 忠诚守护，群体安稳 | 畜牧、看护、团队合作   |
| **蛇蚓潜藏** | 翼火蛇（蛇） + 轸水蚓（蚓） | 隐秘相通，地下联动 | 寻找隐藏之物、暗中谋划 |

> ⚠️ **注意**：  
>
> - 并非所有地支六合的禽星都构成“禽星六合”。  
> - **如“子鼠”与“丑牛”**：虚日鼠（鼠）与牛金牛（牛）在禽性上无直接关联，不视为六合。

### 📌 总结

- **三合局**：重在**五行同气**，主**力量叠加**。  
- **六合对**：重在**禽形相配**，主**功能互补**。  
- **勿混淆**：禽星合局 ≠ 地支合局，需以**禽性**为根本。

掌握此表，即可在排盘时快速识别“助力组合”，大幅提升断事精度。
