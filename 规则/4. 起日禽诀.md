> 起日禽诀 
>
> 七元禽星会者稀，虚奎毕鬼翼氐箕。 但将甲子从头数，元元相续报君知。
>
> 
>
>  假如一元甲子起虚，二元起奎，三元毕，四元鬼，五元翼，六元氐，七元箕，今官历所值之宿是也。每元以甲子、己卯、甲午、己酉日，值日之宿为四将是也，每将符头管十五日止。 一元虚张室轸同，二元奎亢胃房中，三毕尾参斗木獬，四鬼女星危月流，五翼壁角娄金狗，六氐昴心觜火猴，七箕井畔牛边柳，此是七元四将头。 
>
> 每一元甲子六十日，分为四将，每一将管十五日翻禽。如一元甲子日虚日鼠值符头管事，戊寅十五日止，至己卯张月鹿管事十五日，至甲午符头起室管事十五日，至己卯符头起轸管事十五日，共六十日为一元甲子是也，余元仿此。



这段口诀是演禽术中最为关键、也最为复杂的部分——**“起日禽”**。

如果说“年禽”是看这一年的运势，“日禽”就是看这一天的具体吉凶。你提供的这段文字其实包含了**“定局”**（日禽属于哪一元）和**“定日”**（这一天具体是哪一宿）两个步骤。

我来为你详细拆解这个“时间密码”：

### 🧩 核心原理：什么是“七元甲子”？

普通的干支纪日是60天一个循环（一个甲子），但二十八宿只有28个。28和60的最小公倍数是**420**。
所以，演禽术设立了**“七元”**（7个甲子）共420天为一个大循环，这样才能保证“日、月、金、木、水、火、土”七政与二十八宿完美对齐，周而复始。

### 📖 第一步：确定“属于哪一元” (七元起头)

> **口诀：** “七元禽星会者稀，**虚奎毕鬼翼氐箕**。但将甲子从头数，元元相续报君知。”

这句是说，每一个“元”的开始（即甲子日），值班的星宿是不同的。你要先知道当天的干支是甲子、己卯、甲午还是己酉（这四个被称为“四将”或“符头”），然后看它属于哪一元。

| -------- | ---------------- | ---------- |
| **一元** | **虚**日鼠       | **虚**     |
| **二元** | **奎**木狼       | **奎**     |
| **三元** | **毕**月乌       | **毕**     |
| **四元** | **鬼**金羊       | **鬼**     |
| **五元** | **翼**火蛇       | **翼**     |
| **六元** | **氐**土貉       | **氐**     |
| **七元** | **箕**水豹       | **箕**     |

### 📖 第二步：确定“四将”与“十五日一换”

> **口诀：** “一元虚张室轸同，二元奎亢胃房中……此是七元四将头。”
> **解释：** “每元以甲子、己卯、甲午、己酉日，值日之宿为四将……每将管十五日。”

为了方便排盘，古人把60天分为4段，每段15天（配合节气）。这四个分界点就是**“四将”**：

1. **一将（符头）：** 甲子日开始
2. **二将（气将）：** 己卯日开始
3. **三将（符头）：** 甲午日开始
4. **四将（气将）：** 己酉日开始

### 🛠️ 实战推演：如何查某一天的日禽？

假设我们要推算**“一元”**里面的日期（比如最近的一个一元甲子日是1996年1月28日）：

1. **定基准：** 这一天是**甲子日**，属于**一元**。
   - 根据口诀，一元甲子日起**虚日鼠**。
2. **顺数15天：**
   - **第1-15天**（甲子日至戊寅日）：都是**虚日鼠**在值班。
3. **换二将：** 到了**己卯日**（第16天），根据口诀“一元……二元奎亢胃房中”，一元的己卯日起**张月鹿**。
   - **第16-30天**（己卯日至癸巳日）：**张月鹿**值班。
4. **换三将：** 到了**甲午日**（第31天），根据口诀，一元的甲午日起**室火猪**。
   - **第31-45天**（甲午日至戊申日）：**室火猪**值班。
5. **换四将：** 到了**己酉日**（第46天），根据口诀，一元的己酉日起**轸水蚓**。
   - **第46-60天**（己酉日至癸亥日）：**轸水蚓**值班。

### 📌 总结流程图

如果你要推算某一天的日禽，步骤如下：

1. **找最近的“符头”**：看这一天往前推，最近的一个“甲子、己卯、甲午、己酉”日是哪一天？
2. **定元序**：确定那个“符头”日属于“七元”中的哪一元（这通常需要查万年历或演禽盘，看当时的七政轮值）。
3. **定起始宿**：根据“七元起头口诀”，知道那个“符头”日是哪一宿。
4. **顺数**：从那个“符头”日的星宿开始，按照二十八宿的顺序（角、亢、氐……），顺着数到你要算的那一天。

**举个例子（简化版）：**
假如现在是**一元**时期，今天是**甲子日**。

- 口诀说一元甲子起**虚**。
- 所以今天是**虚日鼠**。
- 明天是乙丑日，顺着数就是**危月燕**。
- 后天是丙寅日，就是**室火猪**……以此类推。

这段口诀之所以重要，是因为它把“干支日”转化为了“星宿日”，是演禽术排盘（起四课三传）的基础。
