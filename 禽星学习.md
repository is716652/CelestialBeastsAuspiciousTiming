### 4. 五行类象与取用神三法（《二十八宿用神速查表》《传统十三类用神规则》）

当遇到“既不在 13 类、也很难直接类比到某一星宿”的物品时，可以借助 **五行类象法**：

| 五行 | 类象特征 | 对应禽星（举例） |
|:---:|:--------|:----------------|
| 金 | 金属、白色、圆形、坚硬 | 亢、牛、娄、鬼 |
| 木 | 植物、长条、青色、生发 | 角、斗、井、奎 |
| 水 | 液体、黑色、流动、向下 | 箕、壁、参、轸 |
| 火 | 红色、发热、尖锐、向上 | 尾、室、觜、翼 |
| 土 | 黄色、方形、稳定、承载 | 氐、女、胃、柳 |

原文总结了 **取用神三法**：

1. **以日禽为体，看其吞啖**  
   - 当不知道用神是谁时，可以先把当日日禽当成“我”；
   - 观察它“吃谁”（在吞啖表里克谁），那一类被吃的禽，就暗示了“容易受损 / 被遮蔽”的物象类别。

2. **以时禽为用，看落宫环境**  
   - 时禽代表“事体当前状态”；
   - 结合锁泊，看其落在何宫：
     - 落“刀砧”（申、酉）→ 物品可能被切割、压坏；
     - 落“江湖”（亥、子）→ 多在水边、低处；
     - 落“山林”（寅、卯）→ 高处、树木附近。

3. **以“象”归类**  
   - 分析物品的形状、功能、材质、颜色等，
   - 找到最相似的禽星，再结合日禽 / 时禽 / 锁泊来判断。

> 《传统十三类用神规则》在这一部分之前，还讲了一个“朱元家走失侍女”的故事，并给出口诀“走失易知”——核心就是：
> - 先定日宿、时宿；
> - 再按“寻男子 / 寻妇人 / 寻小儿”等类别选用神；
> - 然后看“用神星在十二支中的位置”，根据原文逐条断语，推断人物相貌、去向、环境、报信人等细节。
> 这些细致断语在原卷中保留，工程中也已经写入 `lost_items_rules.json`，你在理解了上面三法之后，再去看具体案例，会比直接通读原文更容易消化。

## 三、排盘体系：四禽与演禽盘式

这一部分对应《2~8 起禽诀》《9. 演禽盘式》《10.x 锁泊文稿》，是“如何把时间变成星盘”的核心。

### 1. 四禽：年禽、月禽、日禽、时禽

四禽的计算，分别由四篇“起禽诀”支撑：

- **年禽（起年禽诀）**
  - 以某一“基准年 + 基准星”作为起点（如 1984 年甲子年 → 虚日鼠），
  - 每年顺数一宿，形成 28 年一大循环。
  - 用于表示“这一年整体的大环境”。

- **月禽（起月禽诀）**
  - 先看当年的年禽属于哪一种七政属性（日/月/金/木/水/火/土），
  - 根据口诀“太阳用角木参头，太阴起室火星取，金心土胃水骑牛”，
  - 决定正月起始宿，再顺数 12 个月。

- **日禽（起日禽诀）**
  - 采用**七元甲子 420 天循环**：
    - 7 元 × 60 天 / 元 = 420 天；
    - 每元再分 4 将，每将 15 天；
    - 每元有一组“四将符头”。
  - 给出一个“基准日”（如 1984-02-02 为一元甲子日，起虚日鼠），
  - 用日期差 → 定位到第几元、哪一将、哪一日 → 得出当日日禽。
  - 这是整个演禽术的**心脏**。

- **时禽（起时禽诀）**
  - 先根据日禽的七政属性，从对应的“子时起始星宿”开始（口诀“日虚月鬼火寻箕，水毕木氐金奎位，土宿还从箕位起”），
  - 再按十二时辰顺数，得到每个时辰的时禽（身禽）。

> 实战中：**年禽、月禽**多用于补充背景；**日禽、时禽**则是三传四课与盘式的主角。

### 2. 三传四课与演禽盘式（《9. 演禽盘式》《12. 禽星赋》）

- **四课**（空间）
  - 一课：日禽  
  - 二课：时禽（我）
  - 三课：翻禽（他）
  - 四课：活曜（变）

- **三传**（时间）
  - 初传：日禽（大环境 / 事情起因）
  - 中传：时禽（当前状态 / 我方立场）
  - 末传：翻禽（发展结果 / 对方反应）

《7. 起翻禽》《8. 起活曜诀》《12. 禽星赋》详细规定了：

- 翻禽：以“四将符头 + 时禽”之间的距离为基础，
  - 先一步步推算“将星 → 时禽落点”，
  - 再逆数回“时位”，得到翻禽。
- 活曜：从“活曜起宿”逆推时禽，再顺推回用时，得到“倒将星”（变禽）。

> 可以把三传想成“剧情三幕”，四课则是“盘中四个关键角色”。

### 3. 天地盘与锁泊十二宫（《9. 演禽盘式》《10. 禽星锁泊、与演禽盘式的分工》《10.1》《10.2》《10.4》）

- **演禽盘式**：
  - 强调星宿在“天盘、地盘、人盘”中的位置关系；
  - 日禽 / 时禽 / 翻禽 / 活曜分别安在不同层级上，形成一个立体结构。

- **锁泊十二宫**：
  - 以十二地支为“宫位”，采用“上南下北”的布局：
    - 顶行：巳 午 未 申（南）
    - 中行：辰 [中宫] [中宫] 酉
    - 下行：寅 丑 子 亥（北）
  - 《10.1》《10.2》给出清晰的图形规范与宫位含义。
  - 《10.4 张良锁泊歌》则是“锁泊格局 + 吉凶断语”的完整表述。

- **分工**（《10. 禽星锁泊、与演禽盘式的分工》）：
  - 演禽盘式：偏重“星宿与我 / 他 / 变”的关系，
  - 锁泊十二宫：偏重“星落何宫、宫位环境与吉凶倾向”。

> 实际操作中，常见做法是：
> 1. 先按起禽诀排出四禽与三传；
> 2. 再把关键星宿（尤其是时禽、翻禽、活曜）安入锁泊十二宫；
> 3. 然后用锁泊歌与其它规则做综合判断。

## 四、三大基础解盘规则与物象体系

这一部分以《11》《12》《13》《14》为核心，是“读象”的基础层。

### 1. 三大基础规则体系（《11. 演禽术“解盘”环节最核心、最基础的三大规则体系》）

3. **组合关系**（三合、六合、刑冲破害等）

- 同时给出一些基础格局的判断：
  - 某星为日禽、某星为时禽时的基本象；

> 在项目中，部分内容已经渗入：

## 五、15.1 ~ 15.9：结构类活断规则（中阶）

15.1~15.9 主要是**盘面结构与星组关系的活断规则**，适合在熟悉基础盘式后进一步研习。

### 1. 15.1 二十八宿地支分布

- 说明：
  - 每一宿在十二地支中的“分布规律”；
  - 哪些地支对应哪些宿，更适合哪类象。
- 用途：
  - 为锁泊十二宫中的“星落何宫”提供更细致的解释；
  - 配合寻物和方位推演使用。

### 2. 15.2 合禽诀

- 讲述“**星宿之间的合化关系**”：
  - 哪些星与哪些星相合时，整体格局变得更吉；
  - 哪些组合仅仅是“相扶持”，并不直接导致吉凶翻转。
- 对解盘影响：
  - 两星同时出现或互相呼应时，不要孤立看，要考虑是否形成“合禽”，从而改变力量布局。

### 3. 15.3 八曜强禽

- 八曜：从二十八宿中抽取一组“力量特别凸显”的星宿；
- 规则说明：
  - 当这些星宿落入特定宫位或与特定星同临时，盘面整体“动能、影响力”会显著增强；
- 适合放在“力量提示”层面使用，作为“哪些位置更容易被放大、被看见”的参考。

### 4. 15.4 禽星入庙

- 类似于其他术数中的“入庙 / 得地”概念：
  - 某星在某宫位是自己的“本位 / 庙地”，
  - 表现为力量纯粹、长处易发挥。
- 常见用法：
  - 判断三传中哪些环节“有根基”，哪些“漂浮不定”；
  - 在项目中已经以“入庙与强势星宿”的标签实现。

### 5. 15.5 禽星四季旺衰 + 昼夜旺衰

- 分两部分：
  - **四季旺衰**：星随季节而有势弱之变；
  - **昼夜旺衰**：配合日禽 / 夜禽体系（《ri_ye_qin》）判断“飞”或“伏”。
- 解盘意义：
  - 同一颗星，在不同季节、昼夜时段，表现力不同，
  - 要和日、时信息、时辰索引一起考虑。

### 6. 15.6 禽星贵人宿

- 指示：
  - 哪些星在盘中可视为“贵人星”；
  - 在日禽 / 时禽 / 翻禽 / 活曜中各自担任什么角色时，容易带来帮助与转机。
- 项目中的实现：
  - YanQinPanShi 页的“贵人宿提示”卡片，
  - 对应一个 JSON 配置，部分内容来自本卷。

### 7. 15.7 七曜贵人禄星

- 从“七曜”（日、月、金、木、水、火、土）角度，讨论：
  - 对应的贵人星与禄星分布；
  - 哪些组合表示“资助 / 俸禄 / 资源高地”。
- 研习价值：
  - 对理解“星体背后的能量来源”有帮助，
  - 但在当前项目中大多尚未完全数据化，仍属文本研习阶段。

### 8. 15.8 禽星十三退气法

- 属于**高风险判断体系**之一，目前项目中以“寻物场景”为主要落地方向：
  - 通过星宿在某些序列中的进退来判断“气机退损与否”；
  - 文稿中给出详细的十三种退气类型与适用场景。
- 鉴于其敏感与复杂程度，项目采用：
  - **分阶段整合**、
  - 放入独立的 `advanced_rules` 规划中，
  - 不直接作为普通研习标签对外展示。

### 9. 15.9 禽星四季魁星方

- 指明每个季节对应的“魁星方位”：
  - 某些星宿在某些方位上被视为“魁星”，象征才华、考试、名誉等；
- 在项目中，部分理念已经被抽象为“**四季方位提示**”卡片，

## 八、专题占：二十八宿气象占法（天时预测）

本章节内容源自《16.专题占 二十八宿气象占法》，主要解决“天时”层面的预测，与占人事成败不同。

### 1. 核心逻辑：三层定阴晴

气象预测不依赖“三传四课”，而是通过当日的时间信息与星宿属性的碰撞：

1.  **日辰 vs 日宿 五行生克**：日辰（干支纳音五行）与当日值宿（禽星五行）的生克关系定大势。
2.  **二十八宿本象属性**：每一宿自带的物理象征（如角木蛟主风，箕水豹主雨）。
3.  **季节调节（四季细化）**：根据春、夏、秋、冬不同的气象特征对星宿表现进行修正。

### 2. 五行生克天象对照表

| 关系 | 天气预测 | 象义说明 |
| :--- | :--- | :--- |
| **日辰克日宿** | 阴晦、多云 | 能量受压制，阳光不显 |
| **日宿克日辰** | 天晴 | 水火相激或金木相制，反主明朗 |
| **日辰生日宿** | 阴天 | 能量泄化为云雾 |
| **日宿生日辰** | 晴 | 能量回补，天朗气清 |
| **五行比和(火)** | 晴 | 烈日当空 |
| **五行比和(金/水)**| 雨 | 寒湿凝结 |
| **五行比和(木)** | 风 | 气流涌动 |
| **五行比和(土)** | 阴晦 | 尘土蒙蔽 |

### 3. 四季星宿气象断语（精选）

同一星宿在不同季节，预测结果完全不同。例如：

*   **春季（角、亢）**：狂风砂石起，主风大。
*   **夏季（鬼、柳）**：天大雨，火土相激之象。
*   **秋季（奎、娄、胃、昴）**：雨风生，金旺生水。
*   **冬季（觜、参、井、鬼）**：雪声倾，水旺成雪。

> **研习心得**：
> 气象占法是演禽术“天文”属性的体现。在实际研习中，可以先观察当日天气，再回看当日日禽与日辰的关系，是极好的练手方式。

     - 而不是直接替代现实决策的“吉凶断语”。

7. **应用阶段：寻物与案例练习**  
   - 结合：寻物用神体系 + 盘式 + 15.x 标签，
   - 多做“历史案例复盘”，而非现实即时判断，
   - 在练习中体会每条规则的“适用边界”。
